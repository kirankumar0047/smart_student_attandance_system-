# Smart Attendance System Using Face Recognition

![developer](https://img.shields.io/badge/Developed%20By%20%3A-Kolla%20KiranKumar-red)
---
```
An intelligent attendance system that uses facial recognition to automatically mark and store student attendance with real-time camera input and sound alerts.
```
**Tech Stack and Libraries Used**
```
- Frontend: Streamlit
- Backend: Python
- face_recognition
- OpenCV
- pygame
- sqlite3
- datetime
```

**Modules**
```
1. register_student.py – Register a new student and capture face images.
2. encode_faces.py – Encode all registered faces.
3. recognize_and_log.py – Detect faces and mark attendance.
4. view_attendance_app.py – View attendance reports using Streamlit.
```
